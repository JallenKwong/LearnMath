### 基本三角函数关系的速记方法 ###

![](image/002.png)

如上图，六边形的六个角分别代表六种三角函数，存在如下关系：

1. 对角相乘乘积为1，即sinθ·cscθ=1； cosθ·secθ=1； tanθ·cotθ=1。

2. 六边形任意相邻的三个顶点代表的三角函数，处于中间位置的函数值等于与它相邻两个函数值的乘积，如：sinθ=cosθ·tanθ；tanθ=sinθ·secθ...
